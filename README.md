# PetOwner Frontend

Mock data ile ilerleyen Expo tabanli mobil uygulama iskeleti.

## Kurulum

```bash
npm install
npm run start
```

## Ilk surum kapsamı

- Ana sayfa
- Topluluk
- Ilan verme
- Mesajlasma
- Profil

Backend hazir oldugunda `src/data/mockData.ts` uzerindeki veriler servis katmanina tasinabilir.
